## CYCLE A — Components & Page Layouts (as specced; verdict outcomes noted)

### A·1 Layout archetypes — APPROVED
Grids: 60/40 split · Sidebar 20/80 ("quote left, CTA + logic right") · Grid Block · 4-column 25×4 ("equal-weight content blocks") · 6-column 16.6%×6 ("high-density data, fee grids, allocation splits") · Full-bleed visual · Centered CTA 60/40. Zone anatomy + 5th zone (optional Quote/Framing overlay). 24-ID page archetype library (S1, C4, R9 [no CTA], F4, RS1, A8, P1, R2, A1, T1, F3, C6…). Mobile ≤720px, ≥16px paragraph spacing.

### A·2 Containers — CONDITIONAL (C strong-lift & E surround-block REJECTED)
Card defaults: light = white + hairline · dark = borderless tint · framed main-tint = highlight role. Modal/floating: neutral fill + mandatory 1px border + shadow (floats), ≤40% viewport, 80% overlay opacity. Panel budget table (per-colour % caps travel with the colour). Disclaimer box = top/bottom 2px borders only. Border semantics: 1px structure (dotted secondary) · 2px alert (dashed caution) · 3px emphasis · double luxury. Light-beside-light needs a border. Substrates: Modal-class + Bone-class are the universal layering bases.

### A·3 Tables — CONDITIONAL (A default + D dense/legal only; B/C rejected)
Header = ink bold caps + bottom rule · zebra neutral tint 5–30% dispersed · slate hairlines ≤10% · dense/legal = Payne's-class panel · bans: gray-on-gray text, red table fills (F4 benchmark tint-row sole exception).

### A·4 Site chrome — APPROVED (net-new)
Registry: canvas chrome + main indicator (working default) / surface bar + pill tabs / anchor-fill bar (Navy + gold) / ink sidebar. Active = Cobalt-indicator precedent; inactive = Modal-fill precedent; sticky sanctioned.

### A·5 Heroes — APPROVED (role-assigned)
Navy full-bleed (100% hero-only licence, white text, accents ≤10%, CTA on white panel, never beside Oxford) = front door · title-zone 20% = default · gradient cover (registry: Navy→Bone covers, Modal→Cobalt dividers, Mint→Fog exits, Custody-Green→Gold compliance, Gold→Steel-Purple quotes; 2 stops, horizontal, no dense text across fade, max 1/page) = section dividers ⚖ QR "single colour only" read as a PPT-export constraint · full-bleed stat = landing moments.

### A·6 Footers — APPROVED (role-assigned)
Bottom/left-aligned, regular weight, meta size, ≥12px margin, never overlaps CTA; fills none/Modal/Gold/Fog tints; CTA bar = soft-tint band ("padded footer" framed-CTA law); legal = ink/slate + Crimson only for must-see disclosure.

### A·7 Forms & states — CONDITIONAL (fix: vertical centering)
**Construction law: text in fields, chips, buttons, tooltips is vertically centered.** Field = neutral fill + 1px border · labels 11–12px · disabled = Light-Gray bg + Soft-Gray text + Dust-Gray accent (net-new composite) · success = Approval chip ≤3% max 2/view · warning = amber + charcoal · hover/tooltip = Ice-Blue-class ("hover zones, tooltips, subtle overlays") · never stack two interactive tones · buttons ≥44/36px, hover −15%.

### A·8 Page rhythm — APPROVED as law
Dense/light alternation · buffer ranking ⚖ Modal-class within page / Steel-class between sections / Payne's-class between bright sections · risk arc completes in 2–3 sections, never risk→positive · emotional sections 1–2 max then bridge · one idea per section.
