03 · The four decision laws from this analysis

How to choose between them, forever

Accent metal is the audience dial. Champagne = hospitality/gratitude (GCC and family-office safest) · Brass/Old Gold = heritage institution (universally safe) · JCJ Gold = optimism/premium — Americas growth surfaces, always beside evidence, never beside governance ("Gold → Team = misdirection").

Main colour is the dialect. Cobalt = analyst/pension ("strategy clarity, analyst trust") · Teal = product/action ("calm action without pressure") · Verdigris/greens = custody/wealth · Steel Blue = diligence ("reduces perceived volatility").

Warm ink is the anti-"ominous" mechanism. The docs warn dark canvases can read "stark… ominous" — every approved combo defeats this with ivory/oat/cream/parchment ink + whitespace. The one cool-ink combo (WB4) trades that warmth for quant neutrality on purpose.

Family → audience. Green-blacks → GCC + desk + the brand signature · Deep navies → universal trust and first contact · Charcoals → desk + analyst (caution: GCC/newcomers) · Lifted slates → newcomers + compliance · Petrol → crypto-quant.
