🎨 JCJ Master Color Guide: Functional Tier System with Complete HEX References
Executive Summary
This guide presents JCJ Investing's 55-color palette organized by functional usage tiers, where colors appear multiple times based on their different applications.
A single color like 🟦 Navy #0B2E4E can be Tier 1 for backgrounds, Tier 1 for text, but Never for data visualization—each with different usage rules and constraints.
 The system answers: "What tier is this color for my specific use case?"
The 5-Layer Color Framework
Based on JCJ Brand Guidelines v13, every design operates across five functional layers:
Background Layer – Sets emotional tone and pacing (70–100% coverage)
Overlay/Fill Layer – Adds visual emphasis and rhythm (20–40% coverage)
Structural Box/Chart Layer – Highlights content blocks and logic paths (15–30% coverage)
Text/Headline/Icon Layer – Communicates message content (variable coverage)
CTA/Action Layer – Drives user behavior and key signals (1–5% coverage)

Data: 🟦 Signal Blue #3C61C5 for main series
Verify contrast ratios meet ADA standards

Verify platform compatibility
Test contrast ratios (4.5:1 minimum)
Consider regional preferences
Ensure emotional transitions are smooth

Confirm specific context matches restriction
Get approval if needed
Use minimal percentages (≤5–10%)
Provide neutral transitions before and after
Test in target market if regionally sensitive
Using Tier 4 Colors
Obtain required approvals (Director/Brand team)
Verify regional acceptance
Use absolute minimum coverage (≤2–3%)
Have Tier 2 alternatives ready
Document usage justification
