JCJ Master Color Guide: Complete Functional Tier System
Executive Summary
This guide presents JCJ Investing's 55-color palette organized by functional usage tiers, where colors appear multiple times based on their different applications. A single color like Navy #0B2E4E can be Tier 1 for backgrounds, Tier 1 for text, but prohibited for data visualization—each with distinct rules and emotional functions.
Core Principle: "What tier is this color for my specific use case?"
The 5-Layer Color Framework
Based on JCJ Brand Guidelines v13, every design operates across five functional layers:
Background Layer – Sets emotional tone and pacing (70-100% coverage)
Overlay/Fill Layer – Adds visual emphasis and rhythm (20-40% coverage)
Structural Box/Chart Layer – Highlights content blocks and logic paths (15-30% coverage)
Text/Headline/Icon Layer – Communicates message content (variable coverage)
CTA/Action Layer – Drives user behavior and key signals (1-5% coverage)

⚠️ Common Mistakes to Avoid
❌ Never Do This (Critical Failures)
❌ Teal #36BBA6 as background → Destroys CTA meaning
❌ ETF Yellow #FFD89A + White text → Invisible (2.1:1 ratio)
❌ Navy #0B2E4E + Oxford Blue #002147 → Too similar, no contrast
❌ Crimson #D44D5C for positive content → Hardwired risk association
❌ Bright colors >5% coverage → Amateur, unprofessional
❌ Multiple CTAs in different colors → Decision paralysis
❌ Gray text on gray backgrounds → Accessibility failure
⚠️ Caution Required (Test First)
⚠️ Purple in conservative markets → Regional resistance
⚠️ Light backgrounds for projection → May wash out
⚠️ ETF Yellow on screens → Rendering varies widely
⚠️ Burnt Orange globally → Cultural sensitivities
⚠️ Ice Blue in print → Often invisible
✅ Always Remember (Best Practices)
✅ Test print colors on actual printer → RGB ≠ CMYK
✅ Check hover states for digital → -15% brightness standard
✅ Verify emotional flow between sections → Use bridges
✅ Maintain 4.5:1 minimum contrast → WCAG AA standard
✅ Use neutral bridges between strong colors → Prevents jarring
✅ Document color decisions → For consistency
✅ Get approval for Tier 3/4 colors → Governance requirement
🔍 Troubleshooting Guide
Problem → Solution Framework
Problem: Colors look different on screen vs print → Diagnosis: RGB/CMYK conversion issue → Solution: Use PDF/X-4 export, embed ICC profiles → Prevention: Always design in CMYK for print materials
Problem: CTA doesn't stand out → Diagnosis: Insufficient contrast or coverage → Solution: Increase whitespace to 3x button size → Verification: 5% maximum coverage rule, AAA contrast
Problem: Text hard to read → Quick Fix: Increase font weight (Regular → Medium) → Better Fix: Check contrast ratio, aim for 7:1+ → Best Fix: Switch to Tier 1 text color for function
Problem: Emotional flow feels jarring → Diagnosis: Missing transition colors → Solution: Add neutral bridge (Modal Gray #E9EDF2) → Map: Review complete journey map for deck type
Problem: Regional team rejects color choices → Check: Regional preference matrix → Adjust: Use regional Tier 1 alternatives → Document: Note regional variant for consistency
Problem: Deck feels monotonous ↑ Add: Strategic accent colors (3-5% max) → Vary: Text hierarchy with 3 font sizes → Rhythm: Alternate dense/light sections
Problem: Too many colors competing ↓ Reduce: Maximum 5 colors per slide → Hierarchy: 70-20-10 rule for coverage ✅ Simplify: Return to Tier 1 base palette
