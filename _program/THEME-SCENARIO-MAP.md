## The four decision laws from this analysis

1. **Accent metal is the audience dial.** Champagne = hospitality/gratitude (GCC/FO-safest) · Brass/Old Gold = heritage institution (universally safe) · JCJ Gold = optimism/premium (Americas growth surfaces; keep beside evidence; never beside governance — "Gold → Team = misdirection").
2. **Main colour is the dialect.** Cobalt = analyst/pension dialect · Teal = product/action · Verdigris/greens = custody/wealth · Steel Blue = diligence.
3. **Warm ink is the anti-"ominous" mechanism.** Every dark family's safety against the "stark/funeral" failure mode rests on ivory/oat/cream/parchment + whitespace; the one cool-ink combo (WB4) intentionally trades warmth for quant neutrality.
4. **Family → audience:** Green-blacks → GCC + desk + brand signature · Deep navies → universal trust/first-touch · Charcoals → desk + analyst (⚠️ GCC/newcomer) · Lifted slates → newcomers + compliance · Petrol → crypto-quant.
