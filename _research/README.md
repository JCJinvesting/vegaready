# _research/ — the Research Loop home

**Convention (Loop L2 in `docs/LOOPS.md`):**

- `PROMPTS/P-XX-<slug>.md` — numbered, self-contained prompts ready to paste into Perplexity (deep research mode). Each prompt header states the exact return filename.
- `RETURNS/R-XX-<slug>.md` — **drop the Perplexity report here** under the stated filename (raw paste is fine — the agent cleans and ingests). Alternatively paste in chat and the agent files it.
- `INDEX.md` — the ledger: every prompt, its status (open → returned → ingested), and what it fed.

**Rules:** returns are raw material (T3 until verified), never published verbatim; numbers reach pages only via the cascade/registry path with value types + source tiers. Note: `_program/RESEARCH/` is a separate, older folder holding the brand meaning-layer corpus — leave it alone; desk/ops research lives here.
